# py-nqs

Solving the quantum many-body problem with artificial neural networks
Giuseppe Carleo, Matthias Troyer

Science  10 Feb 2017:
Vol. 355, Issue 6325, pp. 602-606
DOI: 10.1126/science.aag2302
